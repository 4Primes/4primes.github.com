--- title: "Platform Tan\xC4\xB1mayan JAVA Dili" wordpress_url: http://blog.4primes.com/?p=73 layout: post date: 2011-06-23 22:05:49 +03:00 wordpress_id: 73 --- Merhaba arkadaşlar, bu yazımda sizlere esneklik, verimlilik , taşınabilirlik özellikleri ve sunduğu olanaklarla yazılım geliştiriciler için vazgeçilmez bir araç olan java hakında genel bilgiler vereceğim. Java ilk çıktığında daha çok küçük cihazlarda kullanılmak için tasarlanmıştı.Ancak platformdan bağımsızlığı özelliği sayesinde C ve C++ dan daha üstün ve güvenli bir yazılım geliştirme aracı olduğu fark edilince hemen her yerde kullanılmaya başlanılmıştır.Günümüzde özellikle kurumsal alanlarda ve  mobil cihazlarda son derece popülerdir.  Bu kadar dil arasından neden java’yı seçmeliyim?
  • Bu konuda söyleyebileceğim çok sebep var  ama bunların  en önemlisi platformdan bağımsız olmasıdır. Platformdan bağımsız derken ne demek istediğimi şöyle anlatayım:
Örneğin  C veya C++ dilleriyle bir programın kaynak kodunu yazdığımızda bu programı derleyerek çalıştırılabilir dosyasını elde ederiz.( Windows çalışma ortamında çalıştırılabilir dosyalarının uzantıları exe ya da dll'dir. )Çalıştırılabilir dosyamızı başka bir platforma örneğin Lınux’a  birebir  kopyalasak  dahi orada çalıştıramayız. Bunun sebebi programın  sadece Windows için derlenmiş olmasıdır. Yani bizim çalıştırılabilir dosyamızın içerisinde sadece Window işletim sisteminin anlayabileceği   “makine kodları”  vardır. Programı Linux'da çalıştırmak için kodun Linux için yeniden derlenmesi gerekir. Ancak çoğu zaman kod üzerinde değişiklikler de yapmak gerekir. Java’ da bu tür sorunlar yaşanmaz. Java ile yazılmış bir program ise her platformda çalışır. Bu Java'nın "Bir kere yaz, her yerde çalıştır!"  ilkesini açıklar.
  • Java öğrenmesi kolay bir dildir. Biraz programlama geçmişi olan özellikle C++ gibi nesne tabanlı bir dilde deneyimi olan  kişiler için java öğrenmek çok kısa bir zamanda halledilebilir.Öyle ki aynı işi  yapan Java programının C++ programından yarı yarıya daha kısa bir sürede yazılabileceği söylenir.
  • Java güçlü bir dildir. Sadece küçük programlar için değil büyük araçlar tasarlamak için de kullanılabilir.
  • Java programları çok çesitli güvenlik seviyeleri ile bir çok ihtiyaca cevap verecek şekilde  çalışabilir. Bu özelliği sayesinde Internet bulunan Java appletleri güvenle çalıştırılabilir.
  • Java ile çalışma zamanında ortaya çıkan hatalar çok zor gerçekleşir.Hataların çoğu kodun derlenmesi sırasında ortaya çıkar ve düzeltilir.(Çalışma zamanı ortaya çıkabilecek hatalar:dosya bulunamaması, Internet bağlantısı kurulamaması..vb )
Java’nın çalışma mantığı:
  •  Programcı java kodunu yazar.
  • Yazılan kod bir java derleyici ile derlenir.
  • Sonuçta platformdan bağımsızlığı sağlayan , bytekod adı verilen bir tür makine kodu ortaya çıkar.(Bir kez bytekod oluşturulduktan sonra program tüm işletim sistemlerinde çalışabilir.)
  • Bu bytekod Java Sanal Makinesi(Java Virtual Machine) tarafından adım adım işletilir.
Java ile neler yapabiliriz?       Java sayesinde; animasyonlar, kayan yazılar, web sayfalarında chat, hesap makinesi, oyunlar, şifreleme programları,  küçük internet appletleri, kocaman paket programlar, resim işleme programları, tercüme programları, sipariş sistemleri, yazım kontrol programları (artık dursak iyi olacak :)  ve bunlar gibi daha nice program yapabiliriz.Son derece geniş bir alana hitap eden bu dili öğrenmek için daha fazla beklemeyelim ;)
